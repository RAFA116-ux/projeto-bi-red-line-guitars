 Projeto BI - Red Line Guitars

Esse projeto foi desenvolvido como parte de uma atividade acadêmica, simulando uma empresa de vendas de instrumentos musicais chamada **Red Line Guitars**.

A ideia foi transformar dados simples em informações mais claras e úteis para análise.

Durante o trabalho, organizei uma base de dados no Excel, criei um modelo dimensional (Star Schema) e desenvolvi um dashboard no Power BI com indicadores de desempenho.

📈 Principais análises:

* Receita total
* Quantidade vendida
* Ticket médio
* Clientes ativos
* Crescimento mensal

💡 Esse projeto me ajudou a entender melhor como dados podem ser organizados e usados para apoiar decisões dentro de uma empresa.

Ferramentas utilizadas: Excel, Power BI e DAX.
